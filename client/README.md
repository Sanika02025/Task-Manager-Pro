# Task Tracker Pro

A full-stack MERN Task Management application.

## Features

- User-friendly Dashboard
- Add/Edit/Delete Tasks
- Search Tasks
- Filter by Status
- Sort Tasks
- Priority Levels
- Due Dates
- Categories
- Progress Dashboard
- Responsive UI
- MongoDB Database

## Tech Stack

React
Vite
Node.js
Express
MongoDB
Axios
CSS

## Installation

Backend

```bash
cd server
npm install
npm run dev
```

Frontend

```bash
cd client
npm install
npm run dev
```
